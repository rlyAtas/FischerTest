# Сущности

## Федеральная земля (state)

- код
- название

## Темы вопросов (topic)

- название
- связь с федеральной землей

## Переводы тем вопросов (topic_translation)

- связь с темой
- текст перевода
- связь с языком

## Вопрос (item)

- текст вопроса
- связь с темой

## Варианты ответов на вопрос (answer)

- связь с вопросом
- текст ответа
- отметка правильности

## Языки (language)

- название
- код

## Переводы вопросов (item_translation)

- связь с вопросом
- текст перевода
- связь с языком

## Переводы вариантов ответов (answer_translation)

- связь с вариантом ответа
- текст перевода
- связь с языком

## Пользователи (user)

- email
- федеральная земля
- язык
- дата/время создания
- дата/время последнего изменения

## Коды авторизации/регистрации пользователей (login_code)

- email
- хэш кода
- количество попыток
- дата/время создания
- дата/время окончания действия кода
- дата/время удачного использования кода

## Сессии пользователей (session)

- связь с пользователем
- хэш токена
- дата/время создания
- дата/время окончания
- дата/время последней активности
- дата/время отзыва/аннулирования

## Избранные вопросы пользователя (favorite)

- связь с пользователем
- связь с вопросом
- дата/время добавления

## Тренировка пользователей (training)

- связь с пользователем
- связь с федеральной землей
- вид тренировки
- связь с темой тренировки (если выбрана тренировка по теме)
- дата/время начала
- дата/время окончания

### Виды тренировок 

- **all** тренировка по всем вопросам федеральной земли
- **topic** тренировка по теме
- **favorites** тренировка избранных вопросов
- **exam** эмуляция экзамена

## Вопросы тренировки пользователя и его ответы (training_item)

- связь с попыткой тренировки пользователя
- связь с вопросом
- позиция вопроса
- выбранный вариант ответа
- флаг правильности ответа
- дата/время ответа

## Прогресс пользователя по вопросам (item_progress)

- связь с пользователем
- связь с вопросом
- статус
- дата/время последнего ответа

### Статусы

- **problematic** - проблемный
- **learning** - в процессе
- **mastered** - освоен

## Агрегированный прогресс пользователя (user_progress)

- связь с пользователем
- связь с землей
- вероятность готовности к экзамену
