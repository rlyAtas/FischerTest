# Правила экзамена

## Назначение

Документ описывает правила режима `Эмуляция экзамена` в приложении `FischerTest`.

Экзамен в приложении является учебной симуляцией теоретической части экзамена на получение рыболовного билета. Он используется:
- для самостоятельной проверки знаний пользователя;
- для расчета вероятности успешной сдачи экзамена.

## Область применения

Правила из этого документа относятся только к режиму `Эмуляция экзамена`.

## Экзаменационная конфигурация

Для каждой федеральной земли в системе должна существовать собственная конфигурация экзамена.

Конфигурация экзамена должна включать:
- правила формирования пула вопросов для экзамена;
- критерий успешной сдачи экзамена.

## Конфигурация для `NRW`

Формирование пула вопросов:
- экзамен содержит **60** вопросов;
- из каждой темы в экзамен должно попасть по **10** вопросов;
- для каждой темы вопросы выбираются в следующем порядке:
  1. случайным образом выбираются вопросы со статусом **new**;
  2. если вопросов недостаточно, случайным образом выбираются вопросы со статусом **problematic**;
  3. если вопросов все еще недостаточно, случайным образом выбираются вопросы со статусом **learning**;
  4. если вопросов все еще недостаточно, выбираются вопросы со статусом **mastered** с самой старой датой последнего ответа;
- каждый вопрос имеет только один правильный ответ.

При досрочном завершении экзамена неотвеченные вопросы считаются неправильными для оценки результата эмуляции.

Критерий успешной сдачи:
- экзамен считается сданным, если правильно отвечено на **45** вопросов;
- в каждой теме должно быть дано минимум **6** правильных ответов.
