# Экраны

## Экран Главная страница неавторизованного пользователя

Краткое описание сервиса
Кнопка **Вход**

## Страница входа по email

Появляется при нажатии на кнопку "Вход" с главной страницы неавторизованного пользователя.
Поле ввода email
Кнопка **Отправить**

## Страница подтверждения email

Появляется при нажатии на кнопку "Отправить" с экрана ввода email.
Поле ввода одноразового кода
Кнопка **Подтвердить**

## Страница первичной настройки

Появляется при первом входе после подтверждения email пользователем.
Поле для выбора земли
Поле для выбора языка интерфейса, этот же язык используется для вспомогательных переводов, если они доступны
Кнопка **Сохранить**

## Экран Главная страница авторизованного пользователя

Краткое описание сервиса
Кнопка **Профиль**
Кнопка **Тренировка**

## Экран Профиль

Поле для изменения земли
Поле для изменения языка интерфейса, этот же язык используется для вспомогательных переводов, если они доступны
Информация о прогрессе пользователя

## Статистика/прогресс теоретической части

Четырехцветная круговая диаграмма: 
- зеленый сектор - количество вопросов со статусом *освоен*;
- желтый сектор - количество вопросов со статусом *в процессе*;
- серый сектор - количество вопросов со статусом *не изучен*;
- красный сектор - количество вопросов со статусом *проблемный*;
Текст об общем уровне подготовленности к экзамену в процентах.

## Экран выбора тренировки/экзамена

Позволяет выбрать режим тренировки.

Кнопки:
- *Все* - тренируем все вопросы, принадлежащие федеральной земле;
- *Тема* - показываем список тем, после выбора темы доступны вопросы этой темы;
- *Избранное* - тренируем избранные вопросы, тренировка доступна, если у пользователя есть избранные вопросы;
- *Эмуляция экзамена*

## Страница список тем

Показывается при нажатии на кнопку "Тема" на экране выбора тренировки/экзамена.
Отображает список тем для данной земли, которые есть в базе данных.
Немецкое название показывается всегда, перевод показывается дополнительно при наличии.

## Экран Страница вопроса для тренировки/экзамена

Текст вопроса с вариантами ответов на немецком языке.
Переводы текста вопроса и вариантов ответа на язык пользователя, если язык пользователя не немецкий и это не экзамен.
Кнопки перехода к следующему/предыдущему вопросу, если осуществимо.
Прогресс текущей тренировки - количество всего вопросов, количество отвеченных.
Возможность добавить/удалить вопрос в избранное / из избранное.
После выбора ответа система показывает правильный ответ, если это не экзамен.
Кнопка "закончить тренировку/экзамен"

## Экран Окончание тренировки/экзамена

Если это эмуляция экзамена:
- сообщить о результате экзамена: сдан или нет;
- если не сдан, то пояснить причину.
Дать статистику: количество правильных ответов, количество неправильных ответов.
Текст об общем уровне подготовленности к экзамену в процентах.

## Privacy policy

Публичная страница, доступная без авторизации.

Страница должна описывать:
- кто является оператором приложения и ответственным за обработку данных;
- какие персональные данные хранит приложение:
  - email пользователя;
  - выбранную федеральную землю;
  - выбранный язык интерфейса;
  - данные сессий;
  - историю тренировок и эмуляций экзамена;
  - ответы пользователя;
  - прогресс по вопросам;
  - избранные вопросы;
- зачем эти данные используются:
  - для входа по email-коду;
  - для сохранения прогресса;
  - для работы тренировок и эмуляции экзамена;
  - для синхронизации данных между устройствами;
  - для обеспечения безопасности сессий;
- где хранятся данные;
- как долго хранятся данные;
- какие внешние сервисы используются, если они есть:
  - email-провайдер;
  - хостинг;
  - база данных;
  - аналитика;
  - мониторинг ошибок;
- использует ли приложение cookies, local storage или другие механизмы хранения в браузере;
- какие права есть у пользователя в отношении персональных данных;
- как пользователь может запросить удаление аккаунта и связанных персональных данных.

На странице должен быть указан контактный email для вопросов о персональных данных и запроса удаления аккаунта.

Юридический текст Privacy policy должен быть подготовлен или проверен отдельно перед production-запуском.

## Impressum

Публичная страница, доступная без авторизации.

Страница должна содержать контактные и юридические сведения об операторе приложения:
- имя физического лица или юридическое название компании;
- почтовый адрес;
- контактный email;
- при наличии - телефон;
- при наличии - регистрационные данные компании;
- при наличии - VAT ID / Umsatzsteuer-ID;
- ответственное лицо за содержание страницы, если это требуется для проекта;
- ссылку на Privacy policy.

Impressum должно быть доступно из публичной части приложения и из авторизованной части приложения.

Фактические реквизиты оператора должны быть заполнены перед production-запуском. До заполнения реальных реквизитов страница не должна использоваться как юридически готовое Impressum.
